# 🌐 Personal Portfolio Website

This is my personal portfolio website, designed and developed to showcase my skills, projects, and professional journey as a developer. The website reflects my passion for modern web development, clean UI design, and continuous learning in emerging technologies.

## 🚀 About the Project

The portfolio serves as a digital resume where visitors can explore my technical skills, educational background, projects, and contributions. It is built with a focus on responsiveness, performance, and user experience, ensuring seamless access across all devices.

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, animations, and responsive design
* **JavaScript** – Interactivity and dynamic features

## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Interactive typing animation in the hero section
* Smooth scrolling navigation
* Animated cards and hover effects
* GitHub contribution snake integration
* Contact form UI for user interaction
* Clean and modern UI with glowing effects

## 📂 Sections Included

* Home / Hero Section
* Skills & Experience
* Education Timeline
* Projects Showcase
* Open Source Contributions
* Extra Curricular Activities
* Contact Section

## 🎯 Purpose

The main goal of this portfolio is to present my abilities in frontend development, demonstrate my projects, and create a strong online presence for career opportunities and collaborations.
